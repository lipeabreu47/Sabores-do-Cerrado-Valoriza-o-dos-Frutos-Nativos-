# 🌐 Projeto Acadêmico - Site Integrado (ADS)

Este repositório contém o código-fonte de um site responsivo desenvolvido como projeto prático para o curso de Análise e Desenvolvimento de Sistemas. 

## 🧠 Processo de Desenvolvimento & Pesquisa
O projeto não foi apenas código; ele envolveu uma etapa prévia de levantamento de materiais e discussões em equipe. No link abaixo, apresentamos o vídeo detalhando a nossa **roda de conversa**, as pesquisas realizadas e todo o material que serviu de base para o desenvolvimento do site.

## 🚀 Tecnologias Utilizadas
- **HTML5:** Estruturação e semântica.
- **CSS3:** Estilização e layout responsivo.
